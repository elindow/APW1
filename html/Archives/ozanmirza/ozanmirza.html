Charity Finder

This is a website that lets the user search for a Charity through it's finances. The way it works is where the user fills out a series of prefrences presented to them, and then the website sends that data to the GuideStar data center, which returns a list of matching charities, and provides a whole profile of the charity aswell. The agreement between the service and the website ceased, so now the data is not available anymore.
