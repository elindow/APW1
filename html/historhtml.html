HTML Template

This page is to be used as a template for an HTML page. Copy this code (you can only see all of the code if you are viewing this in the repository, on GitHub.com; not if you are viewing this as a web page). Then, make a new document in the HTML folder (be sure to name it with the ".html" extension), paste the code you copied and Commit (save) your new page. Then edit the page for content (describe and link to your Markdown page) and for formatting. As instructed in this code, you will make a new .css (Cascading Style Sheet, or Style Sheet) file in order to modify the formatting of your web page. Your style sheet, combined with HTML tags, is what determines the look of your web page.
